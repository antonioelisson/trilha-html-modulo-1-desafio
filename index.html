Tag ol, ul e li

As tags ol, ul e li são usadas para criar listas. A tag ol para listas com uma ordem específica e ul para listas se ordem. Nos dois tipos é usada a tag li como filha para adicionar os itens de lista.

Tag sub e sup

A tag sub define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O.

Já a tag sup define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW [1]
