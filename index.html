Tag ol, ul, dl e li

As tags ol, ul e li são usadas para criar listas. A tag ol para listas com uma ordem específica e ul para listas sem ordem. Nos dois tipos é usada a tag li como filha para adicionar os itens de lista.

Tag sub e sup

A tag sub define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas.

Exemplo:

H2O.

Já a tag sup define texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé.

Tag blockquote

A tag blockquote especifica uma seção citada de outra fonte.

Os navegadores geralmente recuam o elemento blockquote.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Autem corporis, repellat nobis deleniti exercitationem vero beatae laborum maiores. Ipsam aliquam dolores veniam praesentium eius! Fuga tenetur quo reprehenderit hic natus?
